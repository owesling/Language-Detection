# Neural Network for Language Detection from Audio

## Team members
Owen Wesling (owesling), Emanuel Borror (EmaneulBorror)

## Project decription

A language detection model based on neural networks to identify the language spoken out of ## different languages. This model will be built on datasets of audio files containing the most commonly spoken words in each respective language. Possible data sets of individual keywords can be taken from MLCommons Multilingual Spoken Words, LanguaTalk 5k Most Common Words, and Oxford Languages Audio Data.

The goal is to identify and print out the language that is spoken in an audio file of a sentence. The neural network will determine if words in the tested audio file contains any of the "most common words" of certain languages. Early iterations will detect from languages that are distinctly different from each other. Success of the model will allow more languages to be added into detection that are similar to previously added.
